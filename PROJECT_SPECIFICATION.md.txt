PROJECT NAME

DevOps Developer Portal for Amazon Seller Dashboard

OBJECTIVE

Build a full-stack enterprise DevOps dashboard that monitors and manages the existing Amazon Seller Dashboard backend application.

This is NOT a Kubernetes platform.

This is NOT a microservices architecture.

This is NOT an actual cloud deployment platform.

This is a centralized DevOps portal that monitors backend modules of the Amazon Seller Dashboard application.

The Amazon Seller Dashboard backend already exists.

Existing modules:

- Authentication
- Product
- Inventory
- Orders
- Supplier
- Revenue
- Notifications

These modules already have APIs.

The DevOps Portal should consume these APIs and monitor them.
